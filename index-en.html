About me

My name is Alice! I’m a web, graphic designer, HTML CSS coder and digital illustrator, based in Saint-Petersburg.

I have over 5 years in graphic design experience. In free time I'm illustrator. I have spent over 10 years of working with the Adobe graphic applications. I have a passion for art and design, specifically how these disciplines can integrate with modern web technology. For this reason I decided to learn HTML CSS and I really like it.

Coupling aesthetic design sensibilities with user experience and coding allows me to build the best possible online experience from concept all the way to code delivery. Utilizing the pre-processing technology LESS and CSS architectural methodology - BEM, I try to produce clean, modular code.

By using Wacom tablet and Adobe applications combined with traditional illustration and artistic techniques I deliver compelling digital illustration.
